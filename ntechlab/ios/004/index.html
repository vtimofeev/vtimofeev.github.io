Установка приложения на IOS:


1. Установить приложение FindFace Security IOS 0.4;

2. Добавить компанию NTECH.LAB в доверенные распространители приложений.

Для этого в настройках устройства зайти в:
Основные -> Управление устройством (В разделе профиль) -> NTECH.LAB (В разделе 'Корпоративная программа')
Выбрать NTECH.LAB - Доверять
